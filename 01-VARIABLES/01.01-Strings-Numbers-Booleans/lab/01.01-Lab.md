**Solution:**

0. **type: string**
- Remove the hyphen from the variable name.
- Put the value of the string in quotes: "Bob" or 'Bob'.

```js
    let firstName = 'Bob';
    console.log('0.', 'firstName', firstName, typeof(firstName));
    // 0. firstName Bob string
```
